Plugin Manager

Manage installed plugins. Place plugins in the plugins/ directory.

Install Plugin

Create a directory in plugins/<name>/ with a plugin.json manifest.

plugin.json format:

{
  "name": "my-plugin",
  "version": "1.0.0",
  "description": "Does something",
  "enabled": true
}

Optionally add an index.js with hooks: onInit(app), onBuild(builder), onRoute(router, mode).
